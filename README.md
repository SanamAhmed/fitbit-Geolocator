## Developers

You can open this app to add and navigate to a location:

```javascript
import { launchApp } from 'system';

const args = {
	name: 'Some name',
	coords: [48.13194, 11.54944],
};
const uuid = '6aeb6da0-dfb9-40a5-9fcb-160a5b5e3be5'; // from package.json
launchApp(uuid, args);
```
